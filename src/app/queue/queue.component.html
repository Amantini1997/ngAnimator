Queue

The queue is a data structure of the type FIFO where the user can only add elements at the rear (the bottom of the queue) and remove them from the front. Below is a visual demonstration of how a queue works.
To make the example concrete, the queue is composed by people queuing to enter a shop.

The Methods

A queue has 5 public methods:
  • eqnueue(), which add an element to the rear of the collection;
  • dequeue(), which removes and returns the element at the front of the queue;
  • front(), which returns the element at the front of the queue without removing it.
  • size(), which returns the number of elements contained in the queue.
  • isEmpty(), which returns true is the queue is empty, false otherwise.
All this methods can be used in the animation above to play around with the queue.
